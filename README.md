# eyes-detection-
Real-time eye state detection system that triggers an alert when eyes are closed.
A deep learning-based eye state detection system that classifies whether eyes are open or closed using a custom CNN built with TensorFlow/Keras and OpenCV. Designed for drowsiness detection use cases, the system plays an audio alert via winsound when closed eyes are detected. Includes a full training pipeline with image augmentation, Early Stopping, and Learning Rate Reduction callbacks, evaluated with classification reports and confusion matrices.
